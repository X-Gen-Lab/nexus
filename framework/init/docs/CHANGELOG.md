## [1.0.0] - 2026-01-24

### 新增
- ✨ 自动初始化机制 (nx_init)
  - 6 级初始化顺序
  - 链接器段实现
  - 多编译器支持 (GCC, Arm Compiler, IAR)
  - 初始化统计功能
  
- ✨ 启动框架 (nx_startup)
  - 统一启动序列
  - 弱符号覆盖机制
  - 裸机/RTOS 双模式
  - 可配置启动参数
  - 启动状态查询
  
- ✨ 固件信息 (nx_firmware_info)
  - 编译期固件元数据嵌入
  - 版本编码/解码宏
  - 独立链接器段
  - 版本字符串格式化

## 版本说明

### 版本号格式

`MAJOR.MINOR.PATCH`

- **MAJOR**: 不兼容的 API 变更
- **MINOR**: 向后兼容的功能新增
- **PATCH**: 向后兼容的问题修正

### 变更类型

- `新增` - 新功能
- `变更` - 现有功能的变更
- `弃用` - 即将移除的功能
- `移除` - 已移除的功能
- `修复` - Bug 修复
- `安全` - 安全相关修复
