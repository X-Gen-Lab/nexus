简介¶

什么是 Nexus？¶

Nexus 是一个世界级的嵌入式软件开发平台，提供：

  • 统一 API: 跨不同硬件平台的一致接口

  • 可移植性: 一次编写，多平台运行

  • 高质量: MISRA C 合规，经过充分测试的代码

  • 安全性: 为物联网应用内置安全特性

  • 生态系统: 丰富的中间件、云端集成和工具链
