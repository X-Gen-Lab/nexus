Architecture
============

See :doc:`introduction` for architecture overview.

Layer Description
-----------------

**Application Layer**
    User applications built on top of Nexus platform.

**Middleware Layer**
    Common services: logging, shell, configuration, events.

**OSAL Layer**
    OS abstraction for tasks, synchronization, and timing.

**HAL Layer**
    Hardware abstraction for peripherals.

**Platform Layer**
    MCU-specific implementations.
